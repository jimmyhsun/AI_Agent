## AI_Agent
這是一個基於 Python 3.11.9 的專案，適用於 macOS 環境。請按照以下步驟進行安裝與執行。 

## **環境需求**
 - **作業系統**: MacOS 
 - **Python environment 版本**: 3.11.9

## **執行步驟**
套件依賴  pyaudio  
參考網址 ：https://pypi.org/project/PyAudio/  
自己的env參數需填寫  
Mac 需先執行  
```
brew  install  portaudio
```
``` 
pip install -r requirements.txt
```
安裝完後，開始運行，觀看print 的指令運作  
```
python main.py
```
